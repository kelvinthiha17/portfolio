Huffman Coding

C++ DS&A Binary Trees CLI File Compression

This project is a file compression and decompression tool implemented in C++ using Huffman Coding — a well-known lossless data compression algorithm. The approach builds a binary tree based on character frequency, assigning shorter binary codes to more frequent characters to reduce overall file size by ~40% in tests.

How it works

// step 01

The initial string is read from user input or a file and prepared for processing.

// step 02

The frequency of each character is calculated using an unordered_map.

// step 03

Characters are sorted in ascending order of frequency and inserted into a priority_queue for tree construction.

// step 04

Nodes with the lowest frequencies are repeatedly merged to build the Huffman tree, where each leaf represents a character.

// step 05

The tree is traversed using 0s and 1s to decode each character. For example, to decode 'D', follow 1 → 0 → 1 down the tree.

This project deepened my understanding of tree-based data structures, priority queues, and file I/O in C++. It was a strong exercise in translating algorithm theory directly into working, optimized code.

Technologies: C++, priority_queue, unordered_map, binary tree structures, file I/O.
