Huffman Coding (C++)

This project is a file compression and decompression tool implemented in C++ using Huffman Coding — a well-known lossless data compression algorithm. The approach builds a binary tree based on character frequency, assigning shorter binary codes to more frequent characters to reduce file size.

Step 1: Initial string is prepared from user input.

Step 2: The frequency of each character is calculated.

Step 3: Characters are sorted in ascending order of frequency for tree construction.

Step 4: Nodes with the lowest frequencies are repeatedly merged to build the Huffman tree.

Step 5: The tree is traversed using 0s and 1s to decode each character. For example, to decode 'D', we follow 1 → 0 → 1.

This project deepened my understanding of tree-based data structures like binary trees, and file handling in C++.

Technologies Used: C++, Data structures like priority_queue, unordered_map, file I/O, binary tree structures
