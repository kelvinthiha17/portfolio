Traffic Signal FSM

Verilog TTL Logic Hardware Finite State Machine PCB

This project is a custom-built traffic signal controller implemented using a TTL-based finite state machine. Designed on a physical PCB, the system uses combinational and sequential logic to simulate red, yellow, and green traffic light transitions. A clock pulse triggers state changes, mimicking real-world intersection behavior.

The FSM cycles through states — Red → Green → Yellow → Red — driven by a custom clock circuit. Each state transition is encoded in TTL logic gates, with state registers holding the current signal phase between clock edges.

The project reinforced key concepts in digital electronics: timing diagrams, state encoding, logic gate configuration, and the challenge of translating a Verilog state machine into a physical TTL circuit.

This hands-on build bridged the gap between simulation and real hardware, deepening my understanding of system-level control and physical circuit constraints.

Technologies: Verilog HDL, TTL logic ICs, PCB design, finite state machine theory, custom clock circuit.
