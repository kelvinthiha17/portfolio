Traffic Signal State Machine

This project showcases a custom-built traffic signal controller implemented using a TTL-based finite state machine. Designed on a physical PCB, the system uses combinational and sequential logic to simulate red, yellow, and green traffic light transitions. A clock pulse triggers state changes, mimicking real-world intersection behavior.

The project helped reinforce key concepts in digital electronics, such as timing diagrams, state design, and logic gate configuration. This hands-on experience deepened my understanding of hardware implementation and system-level control, bridging theory and practice.
