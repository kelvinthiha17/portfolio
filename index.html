Hardware + software, both sides

I designed a 16-bit RISC processor from scratch in Verilog — full instruction set, datapath, and control logic. I also built a TTL-based traffic signal state machine on a custom PCB, from schematic to soldered board. On the software side, I write in C++, Java, and Python, and I've built a compiler front-end including a recursive-descent LL(1) parser with RISC code generation.

My background spans digital logic design, circuit analysis, FPGA development, and embedded C/C++. I work at the intersection of hardware and software because that's where the hard problems actually live.

Open to F-1/CPT-authorized roles.
