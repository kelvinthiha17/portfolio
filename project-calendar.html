Yearly Calendar Generator (Java)

This Java project generates a full calendar for any given year, including accurate leap year calculations and proper weekday alignment. The program dynamically builds each month and organizes days to reflect real-world layouts.

User interface where the year is entered.

Once the user enters a valid year and clicks "Generate Calendar", the app calculates the first day of each month and determines if the year is a leap year. It then displays each month in a structured and readable format.

Generated calendar displaying correct monthly layouts.

Technologies Used: Java, control structures, arrays, modular arithmetic, Swing GUI
