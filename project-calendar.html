Yearly Calendar Generator

Java Swing GUI CLI Leap Year Logic

This Java project generates a full calendar for any given year, including accurate leap year calculations and proper weekday alignment. The program dynamically builds each month and organizes days to reflect real-world calendar layouts.

How it works

Once the user enters a valid year and clicks "Generate Calendar", the app calculates the first day of each month using modular arithmetic and determines if the year is a leap year. It then displays all 12 months in a structured, readable format.
